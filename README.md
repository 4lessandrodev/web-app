# Web App - Template

Este é um template para web app

---
Instale as dependências

```sh

$ yarn

```

Execute o projeto

```sh

$ yarn dev

# ou

npm run dev

```

Acesse o navegador

```sh

$ GET http://localhost:3000

```

Para executar storybook

```sh

$ yarn storybook

```

Para executar os testes

```sh

$ yarn test

```
